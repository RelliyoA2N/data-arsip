###################
Data Arsip
###################

Menampilkan dan mengelola data agar memugahkan kita dalam melakukan sortir data. Dengan website ini kita dapat lebih efisien dalam mengelola arsip data yang sangat banyak. 

*******************
Teknologi
*******************

CodeIgneter 3
MySQL

**************************
Fitur dan Fungsi
**************************

1. Authentication
2. Sording data
3. Search data

*******************
Server Requirements
*******************

1. PHP version: 7.4.9
2. MySQL version: 5.7.39
